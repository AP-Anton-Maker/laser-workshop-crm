# 📑 ТЕХНИЧЕСКОЕ ЗАДАНИЕ (SRS): Laser CRM Enterprise

**Версия документа:** 1.0  
**Продукт:** Система управления лазерной мастерской и ЧПУ-производством.  
**Тип приложения:** Web-приложение (Single Page Application) + Фоновый Worker (VK Bot) + ML Service.

---

## 🏗️ 1. АРХИТЕКТУРА И ТЕХНОЛОГИЧЕСКИЙ СТЕК
**1.1. Бэкенд:** Python 3.9+, FastAPI, SQLAlchemy 2.0 (Async), SQLite (с возможностью миграции на PostgreSQL), Pydantic v2.  
**1.2. Фронтенд:** Vanilla JavaScript (ES6+), HTML5, CSS3 (нативные CSS-переменные, Grid/Flexbox). Принцип Mobile-First.  
**1.3. Инфраструктура:** Docker & Docker Compose (опционально), Nginx (Reverse Proxy + HTTPS/TLS), Systemd для управления демонами на Raspberry Pi/VPS.  
**1.4. CI/CD:** Автоматическое тестирование и деплой через GitHub Actions / GitLab CI.  
**1.5. API:** Наличие полной документации OpenAPI (Swagger UI) по адресу `/docs`.

## 🧮 2. СМАРТ-КАЛЬКУЛЯТОР (МОДУЛЬ ЦЕНООБРАЗОВАНИЯ)
Ядро системы. Серверный расчет стоимости на основе 11 уникальных алгоритмов. Наличие защиты: итоговая цена не может быть ниже `MIN_ORDER_PRICE` (напр. 500 руб), наценка за срочность `* 1.5`.

| Алгоритм | Назначение | Логика расчета |
| :--- | :--- | :--- |
| **Fixed** | Штучный товар (фляжки, жетоны) | `Фиксированная цена * Количество` |
| **Area (cm²)** | Гравировка дерева/шильды | `((Ширина_мм/10) * (Высота_мм/10)) * Базовая_Ставка` |
| **Meter/Thickness** | Резка фанеры/акрила | `Метры_реза * (Базовая_Ставка * (Толщина_мм / 3.0))` |
| **Per Minute** | Долгая гравировка (JPT/MOPA) | `Минуты_работы_станка * Цена_за_минуту` |
| **Per Character** | Текст на кольцах/ручках | `Количество_символов * Цена_символа` |
| **Vector Length** | Промышленная лазерная резка | `Длина_вектора_в_метрах * Базовая_Ставка` |
| **Setup + Batch** | B2B тиражи | `Цена_настройки_станка + (Цена_за_штуку * Тираж)` |
| **Photo Raster** | Растровая гравировка фото | `Площадь * Базовая_Ставка * Коэффициент_DPI` |
| **Cylindrical** | Кружки/Термосы (поворотная ось) | `(Диаметр * 3.14 * Длина/100) * Базовая_Ставка` |
| **Volume 3D** | 3D-клише / глубокая выборка | `Площадь(см²) * Глубина(мм) * Базовая_Ставка` |
| **Complex** | Материал + Резка | `(Площадь_материала * Цена) + (Метры_реза * Цена)` |

## 📋 3. УПРАВЛЕНИЕ ЗАКАЗАМИ (CRM И КАНБАН)
**3.1. Жизненный цикл заказа:** Строгий флоу статусов: `NEW` (Новый) ➔ `PROCESSING` (В работе) ➔ `DONE` (Готов) ➔ `DELIVERED` (Выдан/Отправлен).  
**3.2. Хранение макетов:** Поля для прикрепления внешних ссылок (Яндекс.Диск/Google Drive) на `.cdr`, `.dxf`, `.ai`.  
**3.3. Фильтрация и Поиск:** Живой поиск (Live Search) по ID, имени клиента, описанию. Фильтры по станкам (Diode/MOPA/CO2) и статусам.  
**3.4. Массовые операции:** Выделение чекбоксами нескольких заказов для групповой смены статуса или удаления.  
**3.5. Приоритеты и Дедлайны:** Цветовая индикация срочности (Срочно - красный, Обычный - зеленый). Визуальные алерты при просрочке дедлайна.  
**3.6. История и Комментарии:** Логирование изменения статусов (кто и когда изменил). Внутренние текстовые заметки для мастера, невидимые клиенту.

---

## 🤖 4. ИНТЕГРАЦИЯ VK И ЧАТ-БОТ (ОМНИКАНАЛЬНОСТЬ)
**4.1. Автоматический поллинг / Webhooks:** Фоновый асинхронный процесс (VKBottle) для мгновенного приема сообщений.  
**4.2. Бесшовный UI чата в CRM:** Экран, разделенный на 2 части: слева список диалогов, справа история переписки (до 100 последних сообщений). Без перезагрузки страницы.  
**4.3. Авто-идентификация:** Парсинг Имени, Фамилии и VK ID при первом сообщении. Автоматическое создание карточки клиента в базе.  
**4.4. Умный опросник (Бот):** Автоматические сценарии с кнопками на основе прайс-листа БД (например: "Выберите материал" ➔ "Укажите длину и ширину").  
**4.5. Режим "Отпуск":** Системная настройка, включающая автоответчик с заданным текстом ("Мастер в отпуске до 15.08").  
**4.6. Медиа и Голос:** Поддержка отправки фото (эскизов) прямо из CRM в VK. Распознавание голосовых сообщений клиентов (Транскрибация текста).  
**4.7. Триггерные уведомления:** Бот сам пишет клиенту: *"Ваш заказ #123 готов! Вы можете забрать его..."* при смене статуса заказа на `DONE`.

## 👥 5. КЛИЕНТЫ, МАРКЕТИНГ И ЛОЯЛЬНОСТЬ
**5.1. Карточка клиента:** Агрегация всех данных: VK ID, телефон, email, сумма всех покупок (LTV), количество заказов, история чата.  
**5.2. Система лояльности (Кэшбэк):** При переводе заказа в статус `DELIVERED`, клиенту на внутренний баланс автоматически зачисляется 5% от суммы чека (1 балл = 1 рубль).  
**5.3. Оптовая скидочная сетка:** Авто-скидки при создании заказа: 10 шт (-5%), 20 шт (-10%), 50 шт (-15%), 100 шт (-20%).  
**5.4. Сегментация (RFM-анализ):** Автоматическое присвоение тегов: "Новый", "Постоянный", "VIP", "Спящий".  
**5.5. Промокоды:** Учет промокодов на скидку (% или фикс. сумма) с лимитом активаций.

---

## 🏭 6. СКЛАД И ОБОРУДОВАНИЕ (ERP-МОДУЛЬ)
**6.1. Листовой и Штучный учет:** Списание фанеры/оргстекла в квадратных сантиметрах (cm²), списание фурнитуры (кольца, магниты) поштучно.  
**6.2. Алерты складских остатков:** Автоматическая подсветка материалов, остаток которых упал ниже `min_quantity` (Пора заказывать).  
**6.3. Мониторинг оборудования:** Виртуальные карточки станков. Очередь задач на каждый станок.  
**6.4. Учет моточасов излучателя:** Трекинг времени наработки лазерной трубки (CO2) или волоконного источника. Уведомления о необходимости ТО (чистка линз, замена воды в чиллере).  
**6.5. База поставщиков:** Контакты продавцов фанеры, акрила, фурнитуры.

---

## 📊 7. АНАЛИТИКА, ОТЧЕТЫ И AI
**7.1. Дашборд (Chart.js):** Графики выручки по дням (Line), популярность услуг/материалов (Doughnut), воронка статусов (Bar).  
**7.2. Метрики реального времени:** Вывод KPI: конверсия успешных заказов, динамика среднего чека, чистая прибыль (Выручка минус Себестоимость материалов).  
**7.3. AI-Прогнозирование:** Модуль на `scikit-learn`. Анализ дневной выручки за последние 30 дней и математический расчет прогноза дохода на "Завтра" с определением тренда (Падение/Рост).  
**7.4. Генерация отчетов:** Экспорт данных за период в форматы CSV, Excel, PDF. Сравнение "Месяц к Месяцу".

---

## 🛡️ 8. БЕЗОПАСНОСТЬ, БЭКАПЫ И АДМИНИСТРИРОВАНИЕ
**8.1. Авторизация и Сессии:** Защита всех эндпоинтов по JWT токенам. Пароли хэшируются через `bcrypt`. Авто-разлогинивание при истечении сессии.  
**8.2. Ролевая модель (RBAC):** `Admin` (полный доступ), `Manager` (только клиенты и чаты), `Master` (только смена статусов заказов).  
**8.3. Резервное копирование (Backups):** Создание ZIP-архива (Файл SQLite + CSV таблицы). Сохранение в папку с таймстемпом. Авто-очистка бэкапов старше 30 дней. Кнопка скачивания бэкапа из интерфейса.  
**8.4. Защита API:** Защита от SQL-инъекций (через ORM), экранирование HTML от XSS-атак.  
**8.5. Audit Log (Журнал действий):** Неудаляемая история: кто, когда и что изменил в системе.

**10.3. Микроинтерфейсы:** Использование Toast-уведомлений (всплывашки справа внизу) об успехе или ошибке. Модальные окна поверх затемненного фона.  
**10.4. Навигация и Hotkeys:** Понятное боковое меню с иконками (FontAwesome). Поддержка горячих клавиш (Ctrl+S сохранить, ESC закрыть модалку).  
**10.5. Типографика:** Шрифты семейства Sans-Serif (Inter / Roboto). Отличная читаемость на мобильных устройствах (Адаптив).
